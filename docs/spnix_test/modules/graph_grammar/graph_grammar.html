class Rule¶

class rostok.graph_grammar.node.Rule¶

The class contains a graph object for substitution into the generated graph and the target node which will be replaced by self.graph_insert. The feature of the rule’s terminality is automatically determined. Id’s mean V from graph theory, do not intersect with V from generated graph.

Graph grammar class¶

class GraphGrammar¶

class rostok.graph_grammar.node.GraphGrammar(**attr)¶

A class for using generative rules (similar to L grammar) and manipulating the construction graph. The mechanism for assignment a unique Id, each added node using rule_apply will increase the counter. Supports methods from networkx.DiGraph ancestor class
